Text to Image Synthesis Using DCGAN with PyTorch

Jan 2024 - Feb 2024

Summary

  • Objective: Implemented a PyTorch-based Conditional Generative Adversarial Network (CGAN) for text-to-image synthesis, inspired by the DCGAN (Deep Convolutional Generative Adversarial Network) architecture, to generate images based on textual descriptions.
  • Architecture: The model architecture draws from DCGAN principles, utilizing deep convolutional layers to enhance image generation quality. The CGAN is conditioned on text descriptions to guide the image generation process.
  • Requirements: Dependencies include h5py==3.6.0, numpy==1.21.5, Pillow==10.0.0, and torch==2.0.0 for model training and evaluation.
  • Dataset: Utilized the Caltech-UCSD Birds 200 dataset along with text embeddings provided by Reed Scott et al, to train the CGAN for generating bird images based on textual descriptions.
  • Results: Achieved image generation from text descriptions with the DCGAN-based CGAN, showcasing the model's ability to create images that correspond to provided text inputs.
  • References: Generative Adversarial Text-to-Image Synthesis - The foundational paper guiding this implementation.
