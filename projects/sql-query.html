Research Paper

Automatic SQL Query Generation

May 25, 2023

Summary

  • Designed and developed an optimized Flask-based web app, reducing SQL query creation time by 30% to enable users to build queries in under 20 seconds effortlessly. Implemented performance enhancements for a 20% increase in overall application responsiveness.
  • Led the development of an SQL Query Automation tool} using JavaScript, Flask, and MySQL, cutting query creation time by 90 and allowing users to generate complex queries in less than 20 seconds through a drag-and-drop interface.
  • This tool supports a wide range of functionalities, including various types of joins, aggregate functions, and clauses. Users can customize their queries to suit their needs and preferences with ease.
  • One of the standout features is the instant feedback mechanism. Users can immediately view the output of their queries at the bottom of the user interface, enabling rapid iteration and refinement
