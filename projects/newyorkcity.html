Research Paper

New York City Airbnb Price Prediction

Aug 2023 - Dec 2023

Summary

  • Trained multiple machine learning models, including Linear Regression, Random Forest, Gradient Boosting, and a Neural Network, to predict Airbnb listing prices in NYC.
  • Preprocessed dataset with features like neighborhood, room type, number of reviews, and availability, using encoding and scaling techniques.
  • Random Forest and the Neural Network models performed well, with key predictors including location, room type, and availability.
  • Developed an interactive dashboard to visualize price distribution across neighborhoods, room type trends, and the impact of amenities on pricing.
  • The project provides actionable insights for Airbnb hosts to adjust pricing strategies based on key influencing factors in the NYC market.

  • Machine Learning, Data Preprocessing, Feature Engineering, Model Evaluation, Data Visualization,
