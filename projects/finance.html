Research Paper

Personal Finance PowerBI Dashboard

Jan 2024 - Feb 2024

Summary

  • Objective: Created an interactive PowerBI dashboard to analyze and visualize personal financial data, enhancing the understanding of spending behavior.
  • Data Collection: Consolidated daily spending data from Excel into PowerBI, using manual entry of expenses categorized by date, item, location, and comments.
  • Data Cleaning: Standardized item categories and date formats, removed null entries, and merged multiple Excel sheets for seamless integration.
  • Features: Designed dashboards to answer key questions such as monthly spending, category-wise expenditure, location-based spending, and comparisons of food vs. restaurant costs.
  • Measures and Visualizations: Developed measures for total cost, average expenses, and price ranges. Created dynamic visuals including slicers for date ranges, price ranges, and spending categories.
  • Tools Used: Utilized Microsoft PowerBI Desktop, Python for preprocessing, and Excel for initial data management. Implemented slicers and interactive elements for enhanced user experience.
  • Outcome: Enabled detailed analysis of spending behavior, facilitating better financial planning and informed decision-making.
