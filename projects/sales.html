Jan 2024 - Feb 2024

Summary

  • Sales/Customer/Product Analysis Dashboard: An interactive Power BI dashboard analyzing AdventureWorks' sales, customer, and product data for 2016 and 2017, using SQL Server and PowerQuery for comprehensive insights and dynamic reporting.
  • Interactive Reports: Features multiple interactive reports including Executive Summary, Customer Analysis, and Product Analysis, designed to provide high-level overviews and detailed insights.
  • Data Integration: Utilizes SQL views for clean data extraction and integration from the AdventureWorks data warehouse, combined with budget data from XLS files for a complete analysis.
  • Dynamic Filtering: Provides dynamic filtering options for analyzing sales performance by various dimensions such as year, month, and product attributes, tailored for sales and customer analysis.
  • Real-Time Data: Enables real-time data access and updates through Power BI Service, allowing users to interact with the latest sales and financial data online.
  • Setup and Access: Offers both web-based read-only access via Power BI and full access through Power BI Desktop for in-depth exploration and customization.
