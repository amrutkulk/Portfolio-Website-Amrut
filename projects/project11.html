StyleClothNet A Fusion of Clothing Annotation and Neural Style Transfer

Jan 2024 - Feb 2024

Summary

  • Project Overview: Developed a web application using Flask to perform style transfer on clothes, ensuring the background remains intact.
  • Technologies Used: Leveraged VGG19 for Neural Style Transfer (NST) and Detectron2 with Mask R-CNN for instance segmentation of clothing items.
  • Image Preprocessing: Utilized TensorFlow and Keras to preprocess images and apply style transfer algorithms.
  • Custom Model Training: Trained the Mask R-CNN model on a custom dataset, which included various types of clothing such as shirts, t-shirts, hoodies, and skirts, using annotations from makesense.ai.
  • User Interface: Enabled users to upload images, automatically detect and apply style transfer exclusively to clothing items, while preserving the original background.
  • Deployment: Successfully deployed the application and demonstrated the effective integration of style transfer with precise instance segmentation on clothing items.
